# Debts Endpoints

- GET `/debts`
  - Query: `page`, `limit`, `direction`, `status`, `linkedGoalId`
- POST `/debts`
- GET `/debts/:id`
- PATCH `/debts/:id`
- DELETE `/debts/:id`
- GET `/debts/:id/payments`
- POST `/debts/:id/payments`
- PATCH `/debts/:id/payments/:paymentId`
- DELETE `/debts/:id/payments/:paymentId`
- POST `/debts/:id/settle`
- POST `/debts/:id/extend`
