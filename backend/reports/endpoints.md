# Reports Endpoints

## Finance
- GET `/reports/finance/summary` (range totals)
- GET `/reports/finance/categories` (top categories + share)
- GET `/reports/finance/cashflow` (daily/weekly/monthly buckets)
- GET `/reports/finance/debts` (active debts, due soon)

## Planner
- GET `/reports/planner/productivity` (tasks completed, focus minutes, habit streaks)

## Insights (AI-ready)
- GET `/reports/insights/daily` (context for daily insights)
- GET `/reports/insights/period` (weekly/monthly context)
