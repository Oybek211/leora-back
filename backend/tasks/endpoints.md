# Tasks Endpoints

- GET `/tasks`
  - Query: `page`, `limit`, `status`, `priority`, `showStatus`, `goalId`
- POST `/tasks`
- GET `/tasks/:id`
- PATCH `/tasks/:id`
- PUT `/tasks/:id`
- DELETE `/tasks/:id`
- POST `/tasks/:id/complete`
- POST `/tasks/:id/reopen`
- PATCH `/tasks/:taskId/checklist/:itemId`
- POST `/tasks/check-finance-trigger`
