# Transactions Endpoints

- GET `/transactions`
  - Query: `page`, `limit`, `accountId`, `type`, `categoryId`, `dateFrom`, `dateTo`, `goalId`, `budgetId`, `debtId`
- POST `/transactions`
- GET `/transactions/:id`
- PATCH `/transactions/:id`
- DELETE `/transactions/:id`
- POST `/transactions/transfer`
- POST `/transactions/bulk`
