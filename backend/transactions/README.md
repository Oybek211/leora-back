# Transactions Module

## Purpose
Record income, expenses, transfers, and debt-linked payments. Provides detailed transaction data for list and detail views.

## Related Frontend Screens
- Transactions list: `app/(tabs)/(finance)/(tabs)/transactions.tsx`
- Transaction detail: `app/(modals)/finance/transaction-detail.tsx`
- Quick expense: `app/(modals)/finance/quick-exp.tsx`
- Transaction editor: `app/(modals)/finance/transaction.tsx`
