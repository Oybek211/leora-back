# Goals Endpoints

- GET `/goals`
  - Query: `page`, `limit`, `status`, `goalType`, `financeMode`
- POST `/goals`
- GET `/goals/:id`
- PATCH `/goals/:id`
- DELETE `/goals/:id`
- GET `/goals/:id/stats`
- GET `/goals/:id/tasks`
- GET `/goals/:id/habits`
- POST `/goals/:id/link-budget`
- DELETE `/goals/:id/unlink-budget`
- POST `/goals/:id/link-debt`
- DELETE `/goals/:id/unlink-debt`
- GET `/goals/:id/finance-progress`
