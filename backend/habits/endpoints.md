# Habits Endpoints

- GET `/habits`
  - Query: `page`, `limit`, `status`, `habitType`, `goalId`
- POST `/habits`
- GET `/habits/:id`
- PATCH `/habits/:id`
- DELETE `/habits/:id`
- POST `/habits/:id/complete`
- GET `/habits/:id/history`
- GET `/habits/:id/stats`
- POST `/habits/:id/evaluate-finance`
- POST `/habits/evaluate-all-finance`
